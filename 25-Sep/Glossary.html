Glossary of Web Terms

HTML
The standard language for creating web pages.
CSS
Style sheet language used to describe the look of a webpage.
JavaScript
A programming language that adds interactivity to websites.
Browser
Software used to view and interact with websites (e.g., Chrome, Edge).
URL
The address used to access a resource on the internet.
Server
A computer that delivers web content to browsers.
Repository
A storage location for project files, often using Git and GitHub.
Responsive Design
An approach to web design that makes sites look good on all devices.

  3. Apply CSS for design
  4. Add interactivity with JavaScript

    2. Check responsiveness
  6. Upload project to GitHub
